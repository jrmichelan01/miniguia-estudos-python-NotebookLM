 ## Glossário.
 
 * **VARIÁVEIS**: São como caixas na memória do computador usada para armazenar dados.
 * **LOOPS**: São estruturas que permitem repetir um bloco de código várias vezes, enquanto uma condição for verdadeira.
 * **FUNÇÕES**: São blocos de códigos que realizam uma tarefa específica e podem ser chamados em diferentes partes do programa para evitar repetições.
 * **ORIENTAÇÃO a OBJETOS**: É um jeito de programar que organiza o código usando "Objetos" que representam coisas do mundo real, contendo tanto os dados quanto as ações relacionadas a eles.
 * **IF / ELIF / ELSE (Tradução: Se / Senão Se / Senão): São comandos de decisão.
 * ** IF testa uma condição; se for verdadeira, executa o bloco abaixo
 * **ELIF serve para testar outras condições caso a primeira falhe 
 * **ELSE é o "porto seguro" que executa se nada anterior for verdade
 * **PRINT: Uma função usada para exibir mensagens ou valores na tela
 * **INPUT: Serve para receber dados digitados pelo usuário
